## 🕹️ Available Games

| Game | Platform | Description |
| :--- | :--- | :--- |
| **Snake Game** | `pygame` | The classic hunger-driven reptile adventure. |

## 🛠️ Requirements
Most games require `pygame`. You can install it using:
```bash
pip install pygame
```
